MariaDB重置账号密码

1. 编辑my.cnf配置文件
vim /etc/mysql/mariadb.conf.d/50-server.cnf

在文件尾部添加以下内容

[mysqld]
skip-grant-tables

注：如果该文件中存在 [mysqld]，则直接添加第二行内容即可

2. 重启 MariaDB
3. 进入MariaDB修改root密码
$ mysql -uroot

MariaDB [(none)]> use mysql;

MariaDB [mysql]> UPDATE user SET Password = password('123456') WHERE User = 'root';

MariaDB [mysql]> flush privileges;

MariaDB [mysql]> quit;

4. 移除my.cnf中配置的skip-grant-tables，重启MariaDB
