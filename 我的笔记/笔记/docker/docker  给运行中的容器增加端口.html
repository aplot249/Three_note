docker 给运行中的容器增加端口

1、首先先关闭 运行中的容器  docker stop   容器ID

2、然后 关闭docker 服务 systemctl stop docker

3、 然后去  /var/lib/docker/容器ID/hostconfig.conf 文件，修改PortBindings。修改为 "PortBindings":{"80/tcp":[{"HostIp":"","HostPort":"88"}]}

表示 宿主机的88端口指向docker容器的80端口，

同时修改 config.v2.json   "ExposedPorts":{"80/tcp":{}} 表示暴露容器的80端口。
