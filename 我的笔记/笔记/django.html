django

1、django安装一些依赖

pip install django-cors-headers django-simpleui

2、配置跨域

在settings.py文件中 CORS_ORIGIN_ALLOW_ALL = True

3、关闭simpleui的宣传信息提示 SIMPLEUI_HOME_INFO = False

4、配置pymysql为django的数据库引擎

a.安装pymysql pip install pymysql

b.在__init__.py文件中编写如下：

import pymysql pymysql.install_as_MySQLdb()

c.在settings.py中配置如下：

DATABASES = {    'default': {        'ENGINE': 'django.db.backends.mysql',        'NAME': '***',        'USER': '***',        'PASSWORD': '***',        'HOST': '127.0.0.1',        'PORT': 3306    } }
